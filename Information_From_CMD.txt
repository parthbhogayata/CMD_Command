Information Gathering from Command Line
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1] systeminfo ( get information about system ):
						-> systeminfo
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2] msconfig ( to open Configraion ):
						-> msconfig
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
3] getmac ( to get information about mac addresing ):
						-> getmac
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
4] chkdsk ( to check hask disk for security || Process like scan ) :
						-> chkdsk
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
5] vol ( to check volume of drive ):
						-> vol <drive_name> {vol d: }
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
6] winver ( to identify window version ) :
				-> winver
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
7] msinfo32 ( to show summary of System):
				-> msinfo32
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
8] diskpart ( to enter in diskpart command line interface ):
						-> disk part 
					
						-> list disk { show list of disk }

						-> select disk <disk number > { disk selection }
		
						-> detail disk { detail about disk }
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
