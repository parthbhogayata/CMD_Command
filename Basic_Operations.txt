Run CMD as Administator : 
			this will provide all permission for working background.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1 ] :  dir : 
	list of items in particular directory
	with <dir> -> folder
	without <dir> -> file 
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2] : cd ( change Directory) : 
			cd <path>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
3] : mkdir ( make directory ) : to create a folder
					i >> mkdir <folder_name>
					if you want to allow space in name of folder than write name in "----"
					
					ii >> md <folder_name>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
4] : rmdir (remove directory ) :  to delete any folder if folder is empty

					i >> rmdir <folder_name>
					ii >> rd <folder_name>	
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
5] : rmdir /s (remove directory which contain files) :
					rmdir <folder_name> /s
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
6] : cd / ( direct goes out from all folder ) : 
				c:/adarsh/desktop/demo
				if i wish direct goes to c: drive 

				cd / {backslace} <<command 
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
7 ] cls ( clear Screen ) : 
			cls
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
8] ren ( Rename a File and folder ) :
			ren  old_name.txt  new_name.txt
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
9] copy (copy one file one location to other) : 
				copy <file_name to copy> <folder_path destination path>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
10 ] move	(move file one location to other ):
				move <file_name to copy> <folder_path destination path>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
11 ] del ( delete file ):
		del  <file_name.extension> 

		 >> delete multiple file of same extention
			del  *.extension	
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
